# vi

Token là đơn vị văn bản mà mô hình xử lý. Một từ có thể gồm một hoặc nhiều token, tùy ngôn ngữ và cách mô hình tách văn bản.

Tìm hiểu thêm: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#token).

# en

A token is a unit of text processed by a model. One word can contain one or several tokens depending on the language and tokenizer.

Learn more: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#token).
