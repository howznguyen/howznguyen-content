# vi

Pull request, thường viết tắt là PR, là đề nghị đưa một nhóm thay đổi từ nhánh này vào nhánh khác. Đây cũng là nơi team đọc diff, thảo luận, yêu cầu sửa và quyết định có merge hay không.

Tìm hiểu thêm: [GitHub Docs - Pull requests](https://docs.github.com/en/pull-requests).

# en

A pull request, often shortened to PR, proposes merging a set of changes from one branch into another. It is also where a team reviews the diff, discusses changes, requests updates, and decides whether to merge.

Learn more: [GitHub Docs - Pull requests](https://docs.github.com/en/pull-requests).
