# vi

Regression test kiểm tra xem một thay đổi mới có vô tình làm hỏng hành vi cũ từng chạy đúng hay không. Sửa được bug trước mặt mà làm gãy ba chỗ phía sau thì chưa thể gọi là thắng.

Tìm hiểu thêm: [ISTQB Glossary](https://glossary.istqb.org/).

# en

A regression test checks whether a new change has accidentally broken existing behavior that previously worked.

Learn more: [ISTQB Glossary](https://glossary.istqb.org/).
