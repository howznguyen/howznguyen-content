# vi

Agentic loop là vòng lặp agent quan sát trạng thái hiện tại, chọn hành động, dùng công cụ, đọc kết quả rồi tiếp tục cho tới khi hoàn thành hoặc gặp điều kiện dừng.

Tìm hiểu thêm: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#agentic-loop).

# en

An agentic loop is the cycle in which an agent observes the current state, chooses an action, uses a tool, reads the result, and continues until it finishes or reaches a stopping condition.

Learn more: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#agentic-loop).
