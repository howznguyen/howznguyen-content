# vi

Hallucination, thường gọi là “ảo giác AI”, là khi model tạo ra thông tin nghe rất hợp lý nhưng sai sự thật rồi trình bày như thể nó đúng. Câu chữ càng trôi chưa chắc bằng chứng càng chắc.

Tìm hiểu thêm: [Google Machine Learning Glossary, Hallucination](https://developers.google.com/machine-learning/glossary/generative#hallucination).

# en

A hallucination is plausible-sounding but factually incorrect output that a generative AI model presents as if it were true. Fluent wording is not evidence of factual accuracy.

Learn more: [Google Machine Learning Glossary, Hallucination](https://developers.google.com/machine-learning/glossary/generative#hallucination).
