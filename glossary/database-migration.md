# vi

Database migration là một thay đổi có phiên bản dùng để đưa cấu trúc hoặc dữ liệu của database từ trạng thái cũ sang trạng thái mới. Nó có thể thêm bảng, đổi cột, thêm ràng buộc hoặc chuyển đổi dữ liệu, nên vẫn cần được review và kiểm tra như code.

Tìm hiểu thêm: [Martin Fowler - Evolutionary Database Design](https://martinfowler.com/articles/evodb.html).

# en

A database migration is a versioned change that moves a database schema or its data from one state to another. It can add tables, alter columns, introduce constraints, or transform existing data.

Learn more: [Martin Fowler - Evolutionary Database Design](https://martinfowler.com/articles/evodb.html).
