# vi

Trong các bài viết trên site này, agent là một hệ thống AI có thể dùng công cụ để đọc file, chạy lệnh, sửa code hoặc gọi API, thay vì chỉ trả lời bằng văn bản.

Tìm hiểu thêm: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#agent).

# en

On this site, an agent is an AI system that can use tools to read files, run commands, edit code, or call APIs instead of only replying with text.

Learn more: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#agent).
