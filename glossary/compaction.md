# vi

Compaction là việc rút gọn lịch sử khi context window sắp đầy, thường bằng cách thay phần hội thoại dài bằng một bản tóm tắt. Nó giúp session đi tiếp nhưng một số chi tiết nhỏ có thể bị mất, kiểu dọn bàn xong mới nhớ tờ giấy quan trọng cũng nằm trong đống đó.

Tìm hiểu thêm: [Claude Code Docs - Explore the context window](https://code.claude.com/docs/en/context-window).

# en

Compaction shortens the history when the context window is nearly full, usually by replacing a long conversation with a summary. It lets the session continue, but some smaller details can be lost along the way.

Learn more: [Claude Code Docs - Explore the context window](https://code.claude.com/docs/en/context-window).
