# vi

MCP, viết tắt của Model Context Protocol, là một chuẩn để ứng dụng AI kết nối với công cụ và nguồn dữ liệu bên ngoài qua một giao diện chung.

Tìm hiểu thêm: [MCP - Architecture overview](https://modelcontextprotocol.io/docs/learn/architecture).

# en

MCP, short for Model Context Protocol, is a standard that lets AI applications connect to external tools and data sources through a shared interface.

Learn more: [MCP - Architecture overview](https://modelcontextprotocol.io/docs/learn/architecture).
