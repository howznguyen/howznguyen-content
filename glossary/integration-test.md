# vi

Integration test kiểm tra các phần đã nối với nhau có phối hợp đúng không, thay vì chỉ kiểm tra từng hàm hoặc module riêng lẻ. Ví dụ, test có thể chạy service cùng database thật của môi trường kiểm tra để xem cả luồng có khớp nhau không.

Tìm hiểu thêm: [ISTQB Glossary - Integration Testing](https://istqb-glossary.page/integration-testing/).

# en

An integration test checks whether connected components or systems work together correctly instead of testing each unit in isolation. For example, it may run a service against a real test database.

Learn more: [ISTQB Glossary - Integration Testing](https://istqb-glossary.page/integration-testing/).
