# vi

Prompt là phần đầu vào bạn gửi cho mô hình, có thể là câu hỏi, yêu cầu, ví dụ hoặc bối cảnh. Prompt càng nói rõ mục tiêu và giới hạn thì mô hình càng ít phải tự đoán.

Tìm hiểu thêm: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#prompt).

# en

A prompt is the input you send to a model, such as a question, instruction, example, or context. The clearer its goal and constraints are, the less the model has to guess.

Learn more: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#prompt).
