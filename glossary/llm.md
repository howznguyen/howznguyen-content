# vi

LLM, viết tắt của Large Language Model, là mô hình ngôn ngữ được huấn luyện trên lượng lớn văn bản và code để dự đoán token tiếp theo. Nhờ đó nó có thể tạo nội dung, giải thích và viết code, nhưng không tự đảm bảo mọi câu trả lời đều đúng.

Tìm hiểu thêm: [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course/llm/transformers).

# en

An LLM, short for Large Language Model, is trained on large amounts of text and code to predict the next token. This lets it generate content, explain concepts, and write code, but it does not guarantee that every answer is correct.

Learn more: [Google Machine Learning Crash Course](https://developers.google.com/machine-learning/crash-course/llm/transformers).
