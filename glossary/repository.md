# vi

Repository, thường gọi ngắn là repo, là nơi lưu code, file và lịch sử thay đổi của một dự án. Trong series này, “đọc repo” nghĩa là xem các file và dấu vết liên quan để hiểu project trước khi sửa.

Tìm hiểu thêm: [GitHub Docs - About repositories](https://docs.github.com/en/repositories/creating-and-managing-repositories/about-repositories).

# en

A repository, often shortened to repo, stores a project's code, files, and revision history. In this series, reading the repo means inspecting the relevant files and history before making changes.

Learn more: [GitHub Docs - About repositories](https://docs.github.com/en/repositories/creating-and-managing-repositories/about-repositories).
