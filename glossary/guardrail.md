# vi

Guardrail là quy tắc hoặc cơ chế giúp ngăn hệ thống thực hiện hành động không mong muốn. Với coding agent, guardrail có thể là giới hạn quyền, đường dẫn được sửa, lệnh được chạy hoặc bước bắt buộc phải xin phép.

Tìm hiểu thêm: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#guardrails).

# en

A guardrail is a rule or mechanism that helps prevent a system from taking unwanted actions. For a coding agent, it can restrict permissions, editable paths, executable commands, or steps that require approval.

Learn more: [Google Machine Learning Glossary](https://developers.google.com/machine-learning/glossary#guardrails).
