# vi

Trong phần mềm, secret là thông tin nhạy cảm dùng để xác thực hoặc cấp quyền, như API key, mật khẩu, token hay private key. Nó không nên xuất hiện trong code, log, prompt hoặc commit vì xoá dòng chữ đi chưa chắc đã xoá được quyền đã lộ.

Tìm hiểu thêm: [GitHub Docs - Secret scanning](https://docs.github.com/en/code-security/concepts/secret-security/secret-scanning).

# en

In software, a secret is sensitive authentication or authorization data such as an API key, password, token, or private key. It should not appear in code, logs, prompts, or commits.

Learn more: [GitHub Docs - Secret scanning](https://docs.github.com/en/code-security/concepts/secret-security/secret-scanning).
