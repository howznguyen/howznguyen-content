# vi

Git worktree là một thư mục làm việc bổ sung được gắn với cùng repository, nhờ đó bạn có thể mở nhiều nhánh cùng lúc ở các thư mục riêng. Nó tách vùng file đang sửa, nhưng không tự giải quyết chuyện hai task cùng đụng một quyết định.

Tìm hiểu thêm: [Git - git-worktree Documentation](https://git-scm.com/docs/git-worktree.html).

# en

A Git worktree is an additional working directory attached to the same repository, allowing multiple branches to be checked out at once in separate directories. It separates working files but does not resolve conflicting decisions between tasks.

Learn more: [Git - git-worktree Documentation](https://git-scm.com/docs/git-worktree.html).
