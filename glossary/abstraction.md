# vi

Abstraction là một lớp trừu tượng giúp che bớt chi tiết phức tạp phía sau một cách dùng đơn giản hơn. Nó hữu ích khi thật sự làm code dễ hiểu, nhưng thêm một lớp chỉ để nhìn có vẻ “xịn” hơn thì đôi khi chỉ chuyển sự rối sang phòng khác.

Tìm hiểu thêm: [MDN Glossary - Abstraction](https://developer.mozilla.org/en-US/docs/Glossary/Abstraction).

# en

An abstraction hides complex implementation details behind a simpler interface. It can reduce complexity, but an unnecessary abstraction may only move that complexity somewhere else.

Learn more: [MDN Glossary - Abstraction](https://developer.mozilla.org/en-US/docs/Glossary/Abstraction).
