# vi

Sandbox là môi trường chạy bị giới hạn quyền và tài nguyên để một chương trình không thể tự do chạm vào toàn bộ hệ thống. Trong agent coding, sandbox thường chặn hoặc thu hẹp quyền đọc file, ghi file, dùng mạng và chạy tiến trình.

Tìm hiểu thêm: [NIST CSRC Glossary - Sandbox](https://csrc.nist.gov/glossary/term/sandbox).

# en

A sandbox is a restricted execution environment that limits which resources and permissions a program can access. In agentic coding, it often constrains file access, network use, and process execution.

Learn more: [NIST CSRC Glossary - Sandbox](https://csrc.nist.gov/glossary/term/sandbox).
