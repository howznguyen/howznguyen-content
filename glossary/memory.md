# vi

Trong coding agent, memory là thông tin được lưu bên ngoài lịch sử trò chuyện hiện tại để có thể dùng lại ở session sau. Nó thường giữ sở thích, quy ước dự án hoặc bài học đã rút ra, chứ không phải agent tự nhớ mọi thứ như con người.

Tìm hiểu thêm: [Claude Code Docs - How Claude remembers your project](https://code.claude.com/docs/en/memory).

# en

In a coding agent, memory is information stored outside the current conversation so it can be reused in later sessions. It often holds preferences, project conventions, or learned lessons rather than giving the agent human-like recall of everything.

Learn more: [Claude Code Docs - How Claude remembers your project](https://code.claude.com/docs/en/memory).
