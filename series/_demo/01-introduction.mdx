## What is Next.js?

Next.js is a React framework that gives you building blocks to create fast, full-stack web applications. It handles the tooling and configuration needed for React, and provides additional structure, features, and optimizations.

## Why Next.js?

React is a library for building user interfaces. While powerful, you still need to make decisions about routing, data fetching, and more. Next.js provides:

- **File-based routing** — Pages are defined by the file system
- **Server-side rendering** — Pages can be rendered on the server
- **API routes** — Build your API alongside your frontend
- **Built-in optimizations** — Images, fonts, and scripts are optimized automatically

## Prerequisites

Before starting this series, you should be familiar with:

- HTML, CSS, and JavaScript fundamentals
- React basics (components, props, state, hooks)
- Node.js and npm/yarn/pnpm
